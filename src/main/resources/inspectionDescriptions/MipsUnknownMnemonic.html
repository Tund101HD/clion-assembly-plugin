Reports tokens in instruction position that are not recognized MIPS mnemonics.

The first token of a statement is expected to be an instruction mnemonic, a label, or a directive. This inspection flags an instruction-position token that matches none of the known MIPS mnemonics (real instructions and common pseudo-instructions) — most often a typo.

For example: 


    addd    $t0, $t1, $t2   ; typo for 'add'
    jjr     $ra             ; not a real instruction

Pseudo-instructions accepted by common assemblers (e.g. li, la, move, nop, b) are recognized. If you use a mnemonic the plugin does not yet know, disable this inspection or report it.
