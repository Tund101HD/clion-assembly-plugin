Reports references to symbols that are neither defined in this file nor declared extern.

A symbol used as a jump/call target or operand must resolve to a label, a data definition, an equ constant, or an extern declaration — in this file or, for extern names, another translation unit. Unresolved names produce "symbol not defined" errors at link time.

For example: 


    jmp     nowhere     ; 'nowhere' is never defined and not declared extern

Cross-file references are resolved against other NASM files in the project. If a name is provided by a library or another object file, declare it with extern to silence the warning.
