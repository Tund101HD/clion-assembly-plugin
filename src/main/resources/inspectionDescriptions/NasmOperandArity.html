Reports instructions used with the wrong number of operands.

Each instruction expects a fixed operand count (zero, one, two, or three). This inspection flags statements that supply too few or too many operands for the mnemonic used.

For example: 


    mov     rax             ; 'mov' requires two operands
    ret     rax, rbx        ; 'ret' takes none (or one immediate)
    inc                     ; 'inc' requires one operand

Instructions with optional operands (such as ret with an immediate, or three-operand imul) accept their valid forms without warning.
